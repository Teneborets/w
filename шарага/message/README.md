# Мессенджер - Система авторизации и регистрации

Веб-приложение мессенджера с системой авторизации и регистрации.

## Особенности

- ✅ Регистрация и авторизация пользователей
- ✅ Хранение данных в SQLite базе данных
- ✅ Безопасное хеширование паролей (bcrypt)
- ✅ Минималистичный современный дизайн
- ✅ Адаптивный интерфейс

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте браузер и перейдите по адресу:
```
http://localhost:5000
```

## Цветовая схема

- Основной цвет (#6ECEDA): кнопки, иконки, активные элементы

## База данных

База данных SQLite автоматически создается при первом запуске. Таблица `users` содержит:
- `id` - уникальный идентификатор
- `phone` - номер телефона (уникальный)
- `password_hash` - зашифрованный пароль
- `created_at` - дата создания

## Безопасность

- Пароли хешируются с использованием bcrypt
- Валидация данных на клиенте и сервере
- Защита от SQL-инъекций через параметризованные запросы
